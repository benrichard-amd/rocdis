# rocdis

## POC ROCm dissassembler

Usage:

    rocdis [options] <input file>
    -d                       Disassemble
    -e                       Extract AMDGPU object from x86-64 binary
    -h                       Show usage
    -l                       List kernels
    -k <index>               Disassemble specified kernel
    -o <output file>         used with -e
